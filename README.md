## Overview

Amrutam Assignment is a responsive frontend web application built using **React** and **Vite**.  
The project was developed as part of an assignment to demonstrate component-based architecture, clean UI design, and mobile-first responsiveness.

## Features

- React functional components
- Vite for fast development and builds
- Fully responsive (mobile, tablet, desktop)
- Reusable and modular UI components
- Clean project structure
- ESLint for code quality
  
## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- ESLint

# Clone the repository
git clone https://github.com/SawantAchal/amrutam-assignment.git

# Navigate to project directory
cd amrutam-assignment

# Install dependencies
npm install

# Start development server
npm run dev
